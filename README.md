# Prediksi-Voltage-Listrik
Prediksi Voltage menggunakan model Regresi Linear Berganda dan Time Series
Tujuan utama dari proyek akhir ini adalah:
Menganalisis korelasi antar variabel teknis kelistrikan (seperti Voltage, Global Intensity, dan Sub-metering) guna mengidentifikasi fitur yang paling berpengaruh terhadap konsumsi daya aktif rumah tangga.
Membangun model prediksi dilakukan dengan menggunakan algoritma Regresi Linear Berganda untuk memprediksi nilai Voltage berdasarkan fitur-fitur yang tersedia.
Melakukan pemodelan berbasis time series dengan memanfaatkan variabel Time dan Voltage, sehingga model dapat digunakan untuk memprediksi nilai Voltage berdasarkan dimensi waktu.

Metode 
Algoritma Regresi Linear Berganda:
Alasan Pemilihan: Regresi Linear dipilih karena kemampuannya dalam memodelkan hubungan sebab-akibat antar variabel. Selain itu metode ini sederhana tapi sangat kuat untuk memprediksi nilai berdasarkan angka teknis yang terukur.
Analisis Time Series (Runtun Waktu):
Alasan Pemilihan: Konsumsi energi listrik memiliki karakteristik musiman, tren, dan siklus tertentu yang sangat bergantung pada waktu (jam, hari, atau bulan). Metode Time Series ini khusus digunakan untuk melihat pola kebiasaan dari waktu ke waktu, sehingga kita bisa meramal kebutuhan listrik minggu depan atau bulan depan berdasarkan data masa lalu.
